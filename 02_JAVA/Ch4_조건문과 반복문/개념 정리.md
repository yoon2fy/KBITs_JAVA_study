# Chapter 04 조건문과 반복문
# 4.01 코드 실행 흐름 제어
- **흐름 제어문:** 자바 프로그램은 main() 메소드의 {}을 위에서 아래로 실행하는 흐름.
- **루핑(looping):** 반복문에서 제어문의 처음으로 돌아가 반복 실행하는것.

**table.** 제어문의 종류
| 조건문 | 반복문 |
|-|-|
| if, switch | for, while, do-while |

# 4.02 if 문
``` java
// 첫 번째 방법
if (조건문) {
  실행문;
  실행문;
  ...
}

// 두 번째 방법
if (조건식1){
  } else if (조건식2) {
  } else {
}

// 세 번째 방법
if (조건식){
  } else {
}
```

# 4.03 switch 문
- 먼저, switch문은 **변수의 값**에 따라 **해당되는 case**로 가서 **실행문**을 실행시킨다.
- 만약, **해당되는 case**가 없으면 **default**로 가서 **실행문**을 실행시킨다.

```java
String 변수명 = "변수";

switch(변수명){
  case 값1:
    실행문1st
    break;

  case 값2;
  case 값3;
    실행문3rd
    break;

  default:
    실행문nth
}
```

# 4.04 for 문
- for문이 처음 실행될 때 **초기화식**이 제일 먼저 실행이 된다.
- **조건식**이 TRUE라면 **실행문**을 실행시키고 그 다음 **증감식**이 실행되며 다시 **조건식**을 평가하게 된다.
- 평가 결과가 다시 TRUE라면 **실행문 - 증감식 - 조건식**이 다시 진행이 된다.
- 마지막으로, 평가 결과가 FALSE가 되면, for문이 끝나게 된다. 

```java
for(초기화식; 조건식; 증감식){
  실행문;
}
```

# 4.05 while 문
- while문이 처음 실행 될 때 **조건식**이 TRUE라면 **실행문**을 실행되며 다시 **조건식**으로 돌아가 **조건식**을 평가하게 된다.
- 조건식이 TRUE라면 재실행되며, 조건식이 FALSE라면 while문이 종료가 된다.

```java
while(조건식){
  실행문1;
  실행문2;
}
```

# 4.06 do-while 문
- do-while문은 실행 결과에 따라 반복 실행을 계속할지 결정하는 점에서 while문과 차이가 있다.
- 먼저, **실행문**을 우선 실행한다. **실행문**이 모두 실행되면 **조건식**을 평가하게 된다.
- 조건식이 TRUE라면 **실행문-조건식**을 반복 실행을 하게 되며,
- 조건식이 FALSE라면 do-while문을 종료하게 된다.

```java
do {
  실행문;
} while (조건식);
```

# 4.07 break 문
- break문은 반복문의 실행을 중지하거나, 조건문인 switch문을 종료할 때 사용된다.
- 중첩된 반복문인 경우, break문은 가장 가까운 반복문만 종료하고 바깥쪽 반복문은 종료하지 않는다.
- 만약, 중첩된 반복문에서 바깥쪽 반복문까지 종료시키려면, 바깥쪽 반복문에 이름(Label, 레이블)을 붙이고 'break 이름:'을 사용하면 된다.

# 4.08 continue 문
- continue문은 반복문인 for, while, do-while문에서만 사용된다.
- continue문이 실행되면 for문에서는 **증감식**, while문, do-while문의 **조건식**으로 바로 이동하게 된다.
